# 栈和队列

## 栈

- 基本概念
  - 栈是操作受限的线性表（后进先出）
  - 允许插入和删除的一端称为栈顶，另一端为栈底。
  - 不含有任何元素为空栈
- 顺序存储结构
  - 结构定义（手写）
    - s.top的初始值为-1
    - 栈顶元素s.data[top];
  - 五大操作（手写）
    - 初始化
    - 判空
    - 进栈
    - 出栈
    - 读栈顶元素（不出栈）
  - 共享栈
    - 从一个数组的两端分别存储元素（两个栈分别记作$S_0$和$S_1$）
    - 判空
      - $S_0$空：top0 == 0;
      - $S_1$空：top1 == MaxSize;
    - 判满
      - top1 - top0 == 1;
- 链式存储结构
  - 结构定义（手写）
    - 用不带头节点的单链表
    - 规定所有操作都在表头进行，即表头为栈顶
  - 三大操作（手写）
    - 入栈
    - 出栈
    - 获得栈顶元素



## 队列

- 基本概念
  - 队头出队，队尾入队
  - 先进先出
- 顺序存储结构
  - 结构定义（手写）
    - front：指向队头元素
    - rear：指向队尾的下一个位置
    - 初始：front = rear = 0;
  - 缺点
    - 产生假溢出（q.rear == MaxSize但是还有空间）
- 循环队列
  - 队头指针进一：q.front = (q.front +1) % MaxSize;
  - 队尾指针进一：q.rear = (q.rear + 1) % MaxSize;
  - 队列长度：(q.rear - q.front + MaxSize) % MaxSize;
  - 判空：q.front == q.rear;
  - 判满：
    - 牺牲一个位置，不存元素：（q.rear + 1） % MaxSize == q.front;
    - 在结构中增加size成员
      - 判满：q.size == 0;
      - 判空：q.size == MaxSize;
    - 在结构中增加tag成员，如果是因为删除元素导致q.front == q.rear，则tag = 0；如果是因为增加元素导致q.front == q.rear，则tag = 1。
      - 判满：q.front == q.rear && tag == 1;
      - 判空：q.front == q.rear && tag == 0;
  - 五大操作（手写）
    - 初始化
    - 判满
    - 判空
    - 入队
    - 出队
- 链式存储结构
  - 结构（手写）
    - 带头结点的单链表，表头为队头，表尾为队尾
    - q.front指向头结点
    - q.rear指向队尾元素
  - 四大操作（手写）
    - 初始化
    - 判空
    - 入队
    - 出队

# 矩阵（行列下标都是从1开始）

- 一维数组的地址定位
  - $LOC(a_i) = LOC(a_0) + i \times L$
- 二位数组的地址定位($H_1，H_2$分别为行、列元素个数)（行列下标都是从0开始）
  - 行优先：$LOC(a_{ij}) = LOC(a_{00}) + (i \times H_1 + j) \times L$
  - 列优先：$LOC(a_{ij}) = LOC(a_{00}) + (j \times H_2 + i) \times L$
- 特殊矩阵存储
  - 对称阵：只保存下三角（包括对角线）的元素
  - 三角阵：只保存上三角（下三角）元素
  - 三对角阵：按照行优先的顺序保存在一维数组中
  - 稀疏矩阵：用三元组保存（i，j，value）
